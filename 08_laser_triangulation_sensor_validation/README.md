# Optical Sensing – Laser Triangulation Systems

This project documents engineering work related to laser triangulation sensing systems developed during my time as a Product & Laser Sensor Development Engineer at Julight.

The work focused on validating and characterising optical distance measurement systems based on triangulation principles.

## Key Engineering Concepts Demonstrated

This project demonstrates:

- Optical sensing system validation
- Laser triangulation measurement principles
- Experimental setup design
- Signal-to-noise ratio (SNR) analysis
- Calibration curve generation
- Sensor alignment sensitivity analysis

## System Overview

Laser triangulation sensors measure distance by projecting a laser beam onto a target and detecting the reflected spot position on a position-sensitive detector (PSD). Changes in the position of the reflected spot correspond to displacement of the target.

## My Contributions

- Designed experimental setups to validate laser triangulation sensor performance
- Performed optical alignment and calibration of laser measurement systems
- Characterised measurement accuracy, repeatability, and signal stability
- Analysed PSD output signals and noise behaviour
- Generated characteristic curves describing sensor response across operating ranges
- Supported product validation prior to deployment in industrial applications

## Validation Approach

Typical validation experiments included:

- Controlled target displacement tests
- Measurement linearity verification
- Signal-to-noise ratio analysis
- Optical alignment stability tests
- Environmental robustness checks
